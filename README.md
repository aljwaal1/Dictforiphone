# قاموسي المدرسي PWA

نسخة ويب تعمل على iPhone و Android من المتصفح، ويمكن إضافتها إلى الشاشة الرئيسية.

## التشغيل
افتح `index.html` محلياً أو ارفع الملفات إلى GitHub Pages.

## النشر على GitHub Pages
1. أنشئ مستودع جديد مثل `qamoosi-pwa`.
2. ارفع كل الملفات.
3. من Settings > Pages اختر Branch: `main` و Folder: `/root`.
4. سيظهر رابط التطبيق.

## التثبيت على iPhone
افتح الرابط من Safari ثم Share ثم Add to Home Screen.

## تحديث المحتوى
عدّل الملف:
`assets/data/words.json`
ثم ارفعه إلى GitHub. داخل التطبيق اضغط: الإعدادات > تحديث القاموس من JSON.

## ملاحظات
- لا توجد إشعارات.
- الأصوات خفيفة ويمكن إيقافها من الإعدادات.
- النطق يعتمد على `speechSynthesis` في المتصفح.
