# قاموسي المدرسي PWA

نسخة PWA تعمل على iPhone وAndroid من المتصفح.

## التشغيل
ارفع الملفات كما هي إلى GitHub Pages أو Netlify.

بعد فتح الرابط من Safari على الآيفون:
Share → Add to Home Screen

## المهم في هذه النسخة
- تم إصلاح ربط الأزرار بالكامل بدون `onclick` مباشر.
- القاموس، البطاقات، الاختبار، الملف الشخصي، إضافة كلمة، الإعدادات تعمل من داخل JavaScript موحد.
- الأصوات خفيفة ويمكن إيقافها.
- التحديث يتم من `assets/data/words.json`.

## تحديث المحتوى
عدّل ملف:
assets/data/words.json
ثم ارفعه إلى GitHub، وبعدها من التطبيق اضغط: تحديث القاموس من JSON.
